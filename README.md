# ResumeMatch AI

This is a Next.js application built with Firebase Studio that helps you optimize your resume for job applications using AI.

## Core Features

- **Job & Resume Analysis**: Paste a job description and your resume text (or upload a file) to get started.
- **AI-Powered Matching**: Get a match score to see how well your resume fits the job description.
- **ATS Compatibility Check**: Identify potential formatting issues in your resume that could cause it to be rejected by Applicant Tracking Systems.
- **Optimization Suggestions**: Receive AI-generated ideas to improve your resume's bullet points and keywords.
- **Skill Gap Analysis**: Discover which skills are missing from your resume and get a personalized learning roadmap to acquire them.
- **Application Tracker**: Save every analysis to a personal dashboard. Track the status of each application (Applied, Interviewing, Offer, etc.).
- **AI Insight Engine**: When you update an application's status to "Offer" or "Rejected," the AI analyzes the outcome and provides feedback on what worked and what didn't, helping you improve for future applications.
- **Resume Versioning**: Save and manage multiple versions of your resume directly within the app.

## Getting Started

To run the application locally, first install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:9002`.
