# ResumeMatch AI

This is a Next.js application built with Firebase Studio that helps you optimize your resume for job applications using AI.

## Core Features

- **Job Description & Resume Analysis**: Paste a job description and your resume to get started.
- **AI-Powered Matching**: Get a match score to see how well your resume fits the job.
- **ATS Compatibility Check**: Identify potential issues that could cause your resume to be rejected by Applicant Tracking Systems.
- **Optimization Suggestions**: Receive AI-generated suggestions to improve your resume's bullet points and keywords.

## Getting Started

To run the application locally, use the following command:

```bash
npm run dev
```

The app will be available at `http://localhost:9002`.
