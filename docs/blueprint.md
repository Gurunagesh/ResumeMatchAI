# **App Name**: ResumeMatch AI

## Core Features:

- Job Description Analyzer: Extract key information from job descriptions using AI to identify required skills, experience, and role type.
- Resume Parser: Parse resume content to extract skills, experience, education, and identify potential ATS blockers using AI.
- AI Job-Resume Matching: Provide a match score between a job description and resume, highlighting skill and experience relevance. This is the core tool to evaluate a job fit.
- ATS Compatibility Checker: Analyze resumes for ATS compatibility issues such as formatting problems and keyword density.
- Resume Optimization: Generate suggestions to rewrite bullet points and insert relevant keywords to improve resume effectiveness using AI.

## Style Guidelines:

- Primary color: Deep blue,green colur (#2962FF) to evoke trust and professionalism.
- Background color: Light gray,ranibow colurs (#F5F7FA), a desaturated version of the primary hue.
- Accent color: Electric purple,red colur (#8E44AD) for highlighting key actions and information.
- Headline font: 'Space Grotesk' sans-serif for a tech-forward, modern appearance; body font: 'Inter' for readability.
- Use clean, professional icons to represent skills and job functions.
- Maintain a clean, structured layout with clear section headings and spacing for optimal readability.
